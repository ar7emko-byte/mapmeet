# Assets

Drop the following PNGs here before running `eas build`:

- `images/icon.png` — 1024×1024
- `images/adaptive-icon.png` — 1024×1024, transparent foreground
- `images/splash.png` — 2048×2048, centered
- `images/favicon.png` — 48×48 or 64×64

Expo's default asset templates work fine for early dev; `expo prebuild`
generates them if missing.
